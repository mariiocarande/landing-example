# scrollwave-landing-example

This project uses the following technologies:

- [Next.js](https://nextjs.org) as a framework for building the application.
- [Tailwind CSS](https://tailwindcss.com) as a CSS styling framework.
- [GSAP](https://greensock.com/gsap/) as an animation library.
- [Lenis](https://lenis.js.org) as a library for creating smooth scroll effects.
- [Lucide-React](https://lucide.dev/icons) as an icon library.

## Starting the project

To start the project, run the command `npm run dev` or `yarn dev`.
